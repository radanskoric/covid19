Circle size = number of active cases
Circle colour = daily growth( 0, 50, 200, 500+ new)

This is an interactive visualisation of the spread of Covid-19 pandemic. The size of the circle represents the number of active cases in a location. Radius is a logarithm (base 2) of the number of cases. The color represents new confirmed cases on that day: Green = 0, Yellow = 50, Red = 500+.

This visualisation source code is open sourced (Github Repository), and using the data from Johns Hopkins University Center for Systems Science and Engineering.
